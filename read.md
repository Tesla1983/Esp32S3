# ESP32-S3 基础课程
本课程系统性地介绍嵌入式系统、电子学与底层单片机编程，使用的平台为 ESP32-S3。课程面向初学者，帮助你同时掌握电子学理论基础，以及基于 ESP-IDF 开发真实嵌入式项目的实践能力。

课程共分为 8 讲，内容从基础电学概念逐步进阶到多任务、网络与 ESP32-S3 无线通信。

# 📖 课程大纲

## 第 1 讲 — 基础：电学、电子学与微控制器

本讲从电学与电子学基础出发，逐步引入半导体器件与微控制器核心概念。

### 涵盖主题

- 静电与电荷
- 电压、电流、电阻与功率
- 欧姆定律与电路分析
- 串联与并联电路
- 导体、绝缘体与半导体
- 二极管与晶体管
- 数字信号与模拟信号
- 嵌入式系统简介
- 什么是微控制器？
- ESP32-S3 架构概览

---

## 第 2 讲 — ESP-IDF 平台：环境搭建、IDE 与编程基础

学习如何配置 ESP-IDF 开发环境，并编写你的第一个 ESP32-S3 应用程序。

### 涵盖主题

- ESP32 系列与 ESP32-S3 特性概览
- ESP-IDF 安装与配置
- 向 ESP32-S3 烧录固件
- ESP-IDF 项目结构
- `app_main()` 入口函数
- 变量与数据类型
    - `int`
    - `float`
    - `bool`
    - `char`
- 运算符与表达式
- 流程控制
    - `if / else`
    - `switch`
    - `for`
    - `while`

---

## 第 3 讲 — GPIO、传感器、执行器与电机

学习如何使用 GPIO、ADC、PWM 与驱动模块，让 ESP32-S3 与外部硬件交互。

### GPIO 与模拟特性

- GPIO 输入与输出
- `gpio_set_level()` 与 `gpio_get_level()`
- 上拉与下拉电阻
- ADC 模拟输入
- 使用 LEDC 实现 PWM
- PWM 频率与占空比

### 传感器与输出设备
- LED 与亮度控制
- 蜂鸣器与音调产生
- 按键输入
- 光敏电阻（LDR）
- 火焰传感器
- 避障传感器
- 超声波测距传感器（HC-SR04）

### 电机控制
- 使用 L298N 驱动直流电机
- PWM 调速
- 方向控制
- 舵机控制
- 步进电机基础
- 无刷直流电机（ESC 控制）

---

## 第 4 讲 — 显示与外设

学习 ESP32-S3 与常见显示模块的接口方法。

### 涵盖主题

- 7 段数码管
    - 共阳与共阴
    - 动态扫描（多路复用）
- MAX7219 LED 点阵
- LCD 16×2 显示屏

---

## 第 5 讲 — 通信协议

理解嵌入式系统如何与外设及其他设备通信。

### 通信协议

|协议|速度|连线|常见用途|
|---|---|---|---|
|UART|最高可达 115200+ 波特率|TX/RX|调试、功能模块通信|
|I²C|100–400 kHz|SDA/SCL|传感器、显示模块|
|SPI|最高可达 10+ MHz|MOSI/MISO/SCK/CS|显示器、SD 卡|

### 涵盖主题

- UART 通信
- 串口调试
- I²C 主机通信
- SPI 通信

---

## 第 6 讲 — FreeRTOS 与内存

学习 ESP32-S3 上的多任务机制、任务调度与内存管理。

### 涵盖主题

### FreeRTOS

- 任务与多任务
- 任务优先级
- 延时与时序控制
- 队列（Queue）
- 信号量（Semaphore）
- 互斥锁（Mutex）
- 双核处理基础

### 内存管理

- 栈（Stack）与堆（Heap）
- SRAM 与 Flash 内存
- ESP32-S3 内存区域划分
- 动态内存分配
- PSRAM 基础

---

## 第 7 讲 — Wi-Fi 通信

基于 ESP32-S3 的 Wi-Fi 能力，构建联网型嵌入式应用。

### 涵盖主题

- Wi-Fi Station（STA）模式
- Access Point（AP）模式
- 连接路由器
- TCP/IP 基础
- HTTP 服务器实现
- HTTP 请求方法
    - `GET`
    - `POST`
- JSON 数据处理
- REST API
- Wi-Fi 事件处理

---

## 第 8 讲 — 蓝牙与 BLE

学习如何使用经典蓝牙与 BLE 进行近距离无线通信。

### 涵盖主题

- 经典蓝牙概览
- 低功耗蓝牙（BLE）
- BLE 服务与特征值
- ESP32-S3 BLE 服务端
- ESP32-S3 BLE 客户端
- 通过 BLE 发送传感器数据
- 与移动端应用通信
- 无线设备控制

# 🗺 学习路径

每一讲都建立在前一讲基础之上。

```
[L1] 电学与电子学基础
          ↓
[L2] ESP-IDF 与 C 编程基础
          ↓
[L3] GPIO · 传感器 · 电机
          ↓
[L4] 显示模块与外设
          ↓
[L5] 通信协议
          ↓
[L6] FreeRTOS 与内存管理
          ↓
[L7] Wi-Fi 网络通信
          ↓
[L8] 蓝牙与 BLE
```

- **第 1 讲** 搭建嵌入式所需的电子学基础。
- **第 2 讲** 引入 ESP-IDF 与底层编程概念。
- **第 3 讲** 聚焦硬件接口与控制。
- **第 4 讲** 介绍显示技术与可视化输出。
- **第 5 讲** 讲解设备通信协议。
- **第 6 讲** 解释多任务与内存管理。
- **第 7 讲** 实现可联网嵌入式应用。
- **第 8 讲** 学习蓝牙与 BLE 无线通信。
